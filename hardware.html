Introduction

There are many different peices of hardware used everyday by proffesionals. These include servers, hubs, modems and so on. But, what do they all do? Simple tasks such as sending an email requires wireless access points and the rest of it.

Definitions

Server

A device or program which manages access to a centralized resource or service in a network

Node

A connection point in the network which is a processing device with an assigned address

Network Interface Card (NIC)

Is a card that is installed in a computer so that it can be connected to a network

Switch

A computer networking device that connects devices together on a computer network by using packet switching

Hub

A common connection point for devices in a network

Router

A router is a device that forwards data packets along networks

Bridge

A bridge device filters data traffic at a network boundary

Repeater

A device that regenerates or replicates signals that are weakened by transmissions over long distances

Modem

A device or program that enables a computer to transmit data over (eg. Telephone or cable lines)

Wireless Access Point (WAP)

A hardware device that allows devices to connect through a wireless standard such as Wi-Fi or Bluetooth

What Is a Network?

Introduction

A network is a group of computers, servers or other devices that are connected so they can interact with each other sending information such as data, hardware and software. Any device that is able to connect to a network is known as a network device.

Advantages vs Disadvantages

Advantages:

- Fast Communication

- Can easily share files and data

- Security (can be password protected)

- Speed

Disadvantages:

- Expensive

- Security threats (for large networks)

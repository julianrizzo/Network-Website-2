Protocols

Introduction

Protocols are an official procedure or system of rules. These rules are used when two ore more computers communitcate with each other. These are many different protocols too, such as HTTP, FTP, SLIP etc.

Definitions

IP

Internet Protocol (data is sent from one computer to another on the Internet)

TCP

Transmission Control Protocol (enables two hosts to establish a connection and exchange streams of data)

FTP

File Transfer Protocol (transfer computer files between a client and server on a computer network)

POP

Post Office Protocol (protocol used by local e-mail clients to retrieve e-mail from a remote server)

SMTP

Simple Mail Transfer Protocol (email is delivered from an email client, such as Outlook)

IMAP

Internet Message Access Protocol (stores email messages on a mail server, but allows the end user to view and manipulate the messages)

HTTP

Hypertext Transfer Protocol (defines how messages are formatted and transmitted)

HTTPs

Hypertext Transfer Protocol Secure (secure communication over a computer network)

802.11x Standard

Compare 802.11a, 802.11b, 802.11g, 802.11n and 802.11ac
